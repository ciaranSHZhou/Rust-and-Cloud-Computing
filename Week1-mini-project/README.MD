# Rust CLI: Guess the number
## Description
Rust CLI exercise that runs a "Guess the Number" game  

## Developer Guide
 * $`make format` to format code  
 * $`make lint` to lint  

* Too run, input following command:  
$` cargo run -- play --name "[YOURNAME]"  
For example,  
$`code$ cargo run -- play --name "Joe"` 
* For help  
`cargo run -- --help`
